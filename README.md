Royal Ads System (QBCore)
Royal Ads System არის FiveM სკრიპტი (QBCore ფრეიმვორკისთვის), რომელიც ამატებს რეკლამების განთავსების ინტერაქტიულ და ვიზუალურად დახვეწილ სისტემას თქვენს სერვერზე.

🌟 ფუნქციები
ინტერაქტიული UI: რეკლამები გამოდის სტილიზებულ ქაღალდის ეფექტზე, რაც როლპლეი (RP) გარემოს უფრო მრავალფეროვანს ხდის.

რიგის მართვა: სერვერი ავტომატურად მართავს რეკლამების რიგს, რათა თავიდან იქნას აცილებული ტექსტების გადაფარვა.

მარტივი გამოყენება: მოთამაშეებს შეუძლიათ გამოიყენონ მარტივი ბრძანება /ad.

ეკონომიკაზე დაფუძნებული: რეკლამის გამოქვეყნება ფასიანია (ნაგულისხმევად 15$), რაც ეხმარება სერვერის ეკონომიკას.

ავტომატური იდენტიფიკაცია: სისტემა თავად კითხულობს პერსონაჟის სახელსა და გვარს ხელმოწერისთვის.

🛠 მოთხოვნები
სკრიპტის გამართული მუშაობისთვის საჭიროა:

qb-core

ox_lib

📥 ინსტალაცია
ჩამოტვირთეთ სკრიპტი.

ჩააგდეთ თქვენი სერვერის resources საქაღალდეში (მაგ: [standalone]/royal_ads).

თქვენს server.cfg ფაილში დაამატეთ:

🎮 გამოყენება
